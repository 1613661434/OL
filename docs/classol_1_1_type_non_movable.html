详细描述

不可移动标记类，继承此类的派生类将禁用移动构造和移动赋值 @usage class MyClass : public TypeNonMovable {};

注解
通常与TypeNonCopyable结合使用：class MyClass : public TypeNonCopyable, public TypeNonMovable {};
