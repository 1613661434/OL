template<bool IsDirected = false, bool IsWeighted = false, typename NodeType = int, typename WeightType = int>
class ol::Graph< IsDirected, IsWeighted, NodeType, WeightType >

通用图模板类，支持有向/无向、有权/无权配置 模板参数：

参数
IsDirected是否为有向图（默认 false）
IsWeighted是否为有权图（默认 false）
NodeType节点数据类型（默认 int）
WeightType权重数据类型（默认 int，仅 IsWeighted=true 时有效）

添加一条边（支持有权/无权、有向/无向图的适配）

参数
from边的起点节点
to边的终点节点
args可变参数，仅有权图（IsWeighted=true）时需要传入权重值（类型为WeightType）
注解
1. 若起点/终点不存在，会自动添加节点
  1. 无向图（IsDirected=false）会自动添加反向边（to -> from）
  2. 有权图必须传入权重参数，无权图不可传入额外参数，否则会编译报错
