◆ bucket_sort_float_base()

template<typename RandomIt, typename Compare>

RandomIt last,
size_t num_buckets,
double min_val,
double max_val,
const Compare & comp )

桶排序核心实现（适用于浮点数）

算法逻辑：

  1. 根据数值范围划分桶（桶的划分基于数值大小，与比较器无关）
  2. 将元素分配到对应桶中
  3. 对每个桶内元素使用自定义比较器进行排序
  4. 合并所有桶的结果
