template<typename T>
class ol::BITree< T >

树状数组|二叉索引树（Binary Indexed Tree，BIT）模板类 适用于高效的点更新和前缀和查询操作，时间复杂度均为O(log n)

模板参数
T存储的数据类型（需支持加法运算）
