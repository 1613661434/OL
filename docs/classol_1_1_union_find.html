template<typename T, typename Enable = void>
class ol::UnionFind< T, Enable >

并查集（Union-Find）模板类，用于管理不相交集合的合并与查询

实现了两种存储策略：

  • 整数类型（如int、size_t）使用连续数组存储，效率更高
  • 非整数类型（如std::string、自定义哈希类型）使用哈希表存储，支持动态元素 核心优化策略：路径压缩（加速查询）和按秩合并（控制树高）
    模板参数
    T元素类型，需支持比较操作（整数类型或可哈希类型）
