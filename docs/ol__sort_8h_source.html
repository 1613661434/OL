2/*
3 * 程序名：ol_sort.h
4 * 功能描述：排序算法工具类，提供高效的排序实现，支持多种容器类型与自定义排序规则，特性包括：
5 * - 容器特性萃取：适配STL容器（vector、deque等）和原生数组，统一迭代器操作接口
6 * - 多种排序算法：插入排序、快速排序、希尔排序、冒泡排序、选择排序、堆排序、归并排序等
7 * - 自定义比较器：支持传入符合严格弱序（Strict Weak Ordering）的比较函数/对象
8 * - 提供容器打印功能（调试用），支持所有可范围遍历的容器类型
9 * 作者：ol
10 * 适用标准：C++11及以上（需支持迭代器特性、类型萃取、函数对象等特性）
11 * 核心约束：自定义比较器必须满足严格弱序，需遵守4条规则：
12 * 1. 非自反性：comp(a, a)必须返回false
13 * 2. 非对称性：若comp(a, b)为true，则comp(b, a)必须为false
14 * 3. 传递性：若comp(a, b)与comp(b, c)均为true，则comp(a, c)必须为true
15 * 4. 不可比传递性：若a与b、b与c均不可比（comp(a,b)与comp(b,a)均为false），则a与c不可比
16 * 禁止使用<=、>=等违反严格弱序的逻辑，否则可能导致排序结果错误或运行时未定义行为
17 */

539 std::is_same<typename container_traits<Container>::value_type, std::string>::value,
540 std::vector<Container>>::type
541 radix_group_by_prefix(Container& container, size_t group_pos, int radix = 256)
542 {
543 using traits = container_traits<Container>;
544
545 if (group_pos == 0)
546 throw std::invalid_argument("group_pos must be >= 1");
547 if (radix < 2)
548 throw std::invalid_argument("Radix must be greater than or equal to 2");
549
550 // 调用分组实现
551 auto groups = base::radix_group_by_prefix_base(
552 traits::begin(container),
553 traits::end(container),
554 group_pos,
555 radix);
556
557 // 转换为目标容器类型
558 std::vector<Container> result;
559 result.reserve(groups.size());
560 for (auto& group : groups)
561 {
562 result.emplace_back(group.begin(), group.end());
563 }
564 return result;
565 }
