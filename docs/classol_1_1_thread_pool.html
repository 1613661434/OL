详细描述

template<bool IsDynamic = false>
class ol::ThreadPool< IsDynamic >

线程池模板类，支持动态/固定两种工作模式

模板参数
IsDynamic是否启用动态模式：true为动态扩缩容模式，false为固定线程数模式（默认）
注解
动态模式下会根据任务负载自动调整线程数量，固定模式使用初始化时指定的线程数
线程安全设计，支持多线程并发添加任务
所有线程均通过join模式退出

◆ manager()

template<bool IsDynamic = false>

void ol::ThreadPool< IsDynamic >::manager ( )
inlineprivate

管理者线程主函数（仅动态模式可用）

注解
定期执行以下操作：
  1. 清理已退出的工作线程对象
  2. 根据任务负载和空闲线程数进行扩缩容：
    • 扩容：任务数 > 线程数*2 且未达最大线程数时创建新线程
    • 缩容：空闲线程 > 线程总数的1/2 且超过最小线程数时销毁多余线程
